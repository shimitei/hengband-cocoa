hengband-cocoa
======================
This project is a macOS cocoa frontend of Hengband, one of the most popular  
Angband-clone roguelike game from Japan. Code is greatly inherited from vanilla  
Angband sorce tree.

## Build
    $ cd src
    $ make -f makefile.osx
